# 🔐 Color Image Encryption Using Compressive Sensing and DNA Coding
## 📌 Project Title

A Color Image Encryption Algorithm Based on Compressive Sensing and Block-Based DNA Coding

## 📖Overview

This project introduces a secure color image encryption technique that combines Compressive Sensing (CS) with block-based DNA coding to achieve both data compression and strong encryption.

The algorithm first compresses the image using compressive sensing to reduce storage and transmission requirements. After compression, the image is encrypted using DNA sequence encoding and chaotic algorithms, which significantly improves security.

This hybrid approach produces highly secure encrypted images that are resistant to various cryptographic and statistical attacks, making it suitable for secure image transmission.

## 🎯 Objectives

  Reduce image size using compressive sensing

  Encrypt images using DNA coding techniques

  Improve security using chaotic map randomness

  Protect image data from unauthorized access
 
  Ensure secure transmission of images across networks

# 🧠 System Architecture

The encryption system follows a multi-stage pipeline:

     Input Color Image
        │
        ▼
    Image Compression (Compressive Sensing)
        │
        ▼
    DNA Encoding
        │
        ▼
    Chaotic Map Key Generation
        │
        ▼
    Josephus Pixel Scrambling
        │
        ▼
    DNA Block-Based Encryption
        │
        ▼
    Encrypted Image Output

⚙️ Key Techniques Used:

1️⃣ Compressive Sensing (CS)

Compressive Sensing is used to compress image data before encryption, reducing the amount of data required for storage and transmission.

Benefits:

Lower storage requirements

Faster image transmission

Reduced computational cost

2️⃣ DNA-Based Encryption

In this method, pixel values are converted into DNA sequences using four nucleotides:
     
     | Binary | DNA |
     | ------ | --- |
     | 00     | A   |
     | 01     | T   |
     | 10     | C   |
     | 11     | G   |

3️⃣ Chaotic Map Security

Chaotic systems are used to generate random keys for encryption.

Advantages:

High randomness

Sensitive to initial parameters

Difficult to predict

4️⃣ Josephus Pixel Scrambling

The Josephus algorithm is used to shuffle image pixels, increasing the complexity of the encrypted image.

Benefits:

Breaks spatial correlation between pixels

Improves resistance to statistical attacks

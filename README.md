# encryption
A color image encryption algorithm based on compressive sensing and block based DNA coding.


## Overview

This project presents a novel method for color image encryption by integrating Compressive Sensing(cs) with DNA-based block coding.It compresses the image to reduce storage needs and then encrypts it using DNA sequences and chaotic algorithms.The result is secure,compact image that resists multiple forms of cyberattacks.Its ideal for secure image transfer in sectors like defense,healthcare,and cloud storage.

## Tools and technologies used:

    C#
    visual studio code
    SQL server
    Ml.net
    SOAP
    XML
    IIS server

## Features:

High-compression image encoding using CS

Dynamic DNA-based encryption

use of chaotic maps for randomness and security

pixel scrambling via Josephus algorithm

High resistance to cryptographic and statistical attacks
